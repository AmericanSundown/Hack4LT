Šis projektas atsirado kaip Programavimas Python kalba laisvai pasirenkamo dalyko (dėstomo VU MIF) pagalbinė priemonė bendradarbiavimui su studentais.

Paskaitą dėsto Albertas Gimbutas (el. pašto adresas: albertas.gimbutas @mii.vu.lt)

Programavimas Python kalba (sandas) paskaitos aprašymas:
Python tai aukšto lygio objektinė programavimo kalba, kurią ypatingai lengva išmokti. Python yra galinga ir plačiai taikoma, pvz.: moksliniams skaičiavimams (SciPy), filmų kūrimui (Blender), interneto svetainių kūrimui (Django) ir kt. Šio kurso metu suteikiamos Python programavimo žinios, pažintis su profesionaliais programavimo įrankiais ir praktikomis; išmokoma kokybiškai dalyvauti komandiniuose programavimo projektuose. Python kalba ir jos ekosistema yra atviri, todėl visos kalbos galimybės prieinamos nemokamai komerciniais tikslais. Dėl atvirumo ir aukšto produktyvumo Python yra puikus pasirinkimas kuriant StartUp'us. Kurso pabaigoje organizuojamas viešas hakatonas, kuriame bus galima susipažinti ir pasivaržyti su tikrais Python profesionalais.
