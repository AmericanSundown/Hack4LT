2014-02-10 užduotis (0.5 balo):

  1. Susikurti github.com / bitbucket.org ir stackoverflow.com paskyras.
  2. Prisiregistruoti Hack4.LT puslapyje.
  3. Susikonfigūruoti Linux sistemą darbui
    • Įsidiegti Ubuntu operacinę sistemą. Paruošti interneto prieigą.
    • Instaliuoti VIM, Git ir Mercurial terminale įvykdant komandas: 
      $ sudo apt-get update
$ sudo apt-get install vim
$ sudo apt-get install git-core gitg
$ sudo apt-get install mercurial
$ sudo apt-get install meld kdiff3
    • Susigeneruoti ssh raktą ir jį nurodyti github.com / bitbucket.org puslapiuose prie savo paskyros nustatymų (Account settings -> SSH keys):
      • $ ssh-keygen
$ cat ~/.ssh/id_rsa.pub
    • Nusikopijuoti konfigūracinius VIM, Git, Mercurial ir Bash failus į namų katalogą. 
      $ cd ~/
$ git clone https://github.com/strazdas/home.git
$ mv home/.vim home/.vimrc home/.gitconfig home/.hgrc .
$ mv home/.buildout home/.bashrc .
$ source .bashrc
$ rm -fr home
    • Sugeneruoti sistemos konfigūraciją apibūdinantį failą sys_info 
      $ wget http://hack4.lt/static/files/gen_sys_info.py
$ python gen_sys_info.py
      Su VIM redaktoriumi sys_info faile pakeisti žodžius "checksum" į "kontrolinė suma". Interaktyvus VIM pradžiamokslis. Kai kurios naudotinos VIM komandos. 
      $ vim sys_info
      Gautąjį sys_info failą įkelti čia.
  4. Pasirinkti kokią užduotį įgyvendinsite panaudodami Python. Rekomenduoju rinktis kitos paskaitos laboratorinį darbą. Įgyvendinant šią užduotį reikės parašyti bent 250 Python kodo eilučių. Pasirinktąją užduotį trumpai aprašykite čia.
